# Nadim Calendar

## Objective
A simple Calendar app where people can view and add events

## Install

### Clone the Repo
```shell
git clone https://github.com/nuddin95/ttp2-challenge.git
cd ttp2-challenge
```

### Create the Database
```shell
createdb NadimCalendar
```

### Install and Run
```shell
npm install
npm run start-dev
```
#### Go to the specified local host and you should be at the homepage
